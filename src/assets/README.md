# Assets

Place the following files in this directory:

| File | Description |
|---|---|
| `logo.png` | FELLITO app logo — orange oval with white feather/quill signature mark |
| `fellito-avatar.png` | Fellito character avatar — cyberpunk-style tech consultant illustration |
| `icon.png` | Expo app icon (1024×1024) — use the logo.png scaled/padded |
| `splash.png` | Splash screen (use logo centered on dark `#0A0A0F` background) |
| `adaptive-icon.png` | Android adaptive icon foreground (108×108 safe zone) |
| `favicon.png` | Web favicon (64×64) |

Both `logo.png` and `fellito-avatar.png` were provided by the user directly.
